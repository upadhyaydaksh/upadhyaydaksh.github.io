About Project

  • Pokemon Database is an web application that allows you to get pokemon data from open source api and display it to users.
  • User can select pokemon and details will be shown in the bottom detail section.
  • There is seperate section for checking abilities detail which can view by clicking abilities name in detail section.
  • This app will assist users to check pokemon stats.
