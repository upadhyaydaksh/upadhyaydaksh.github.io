About Project

  • Recipianza is a react native application.
  • Login and signup authentication is done with the help of Firebase.
  • Recipianza contains list of recipies that can viewd offline without the Internet.
  • This application will allow users to mark their favourite recipes to Favourites.
  • User can add and remove recipies from favourites.
  • User can also share recipe on other platforms.
